# Бенчмарк против промышленных продуктов + список улучшений

**Дата:** 2026-06-13
**Цель:** сверить RiskCalc с промышленными системами для банков, инвесткомпаний
и брокеров — какие данные и представления у них есть, чего нет у нас, и какие
улучшения дадут максимум ценности с учётом контекста (рынок РФ, бесплатные данные,
десктоп PySide6, identity-first валидация).

---

## 1. Ландшафт промышленных продуктов

| Категория | Продукты | Что это |
|---|---|---|
| Терминалы данных | Bloomberg Terminal, LSEG/Refinitiv Workspace (Eikon) | рыночные данные, новости, аналитика, чат |
| Risk/Pricing платформы | Bloomberg **MARS**, Murex **MX.3**, **Calypso** (Adenza/Nasdaq), **Numerix**, FIS Adaptiv | front-to-back прайсинг, VaR, XVA, регуляторика |
| Портфельно-риск | BlackRock **Aladdin**, MSCI RiskMetrics/Barra, Bloomberg PORT | факторный риск, атрибуция, what-if, оптимизация |
| Брокерские/исполнение | ION, FlexTrade, Bloomberg AIM/EMSX | OMS/EMS, блоттер, исполнение, позиции в реальном времени |

**Ключевые факты (2025–26):** MARS — единая pricing/data library для market risk,
XVA (CVA/DVA/FVA), SIMM, collateral, hedge accounting, climate ([Bloomberg](https://professional.bloomberg.com/products/risk/mars));
Aladdin мониторит 2000+ риск-факторов ежедневно, 5000 стресс-тестов в неделю,
декомпозиция риска по портфелю/фактору/сектору/бумаге, конфигурируемые
дашборды, AI-комментарии ([BlackRock](https://www.blackrock.com/aladdin/products/aladdin-risk));
MX.3 — полностью переоценивающий движок, real-time P&L/греки/экспозиции, VaR,
стресс, XVA, IM, capital ([Murex](https://www.murex.com/en)).

## 2. Данные: они vs RiskCalc

| Реальное время / стриминг | ✅ tick/intraday | ❌ EOD | **есть** |
| Reference data (corp actions, ISIN, рейтинги) | ✅ полно | 🟡 bondization, дефолт-флаги, дивиденды | частичный |
| Новости / sentiment | ✅ | ❌ | есть (низкий приоритет) |
| Факторные библиотеки (Barra-style) | ✅ | ❌ корреляции из истории | средний |
| Бенчмарк-индексы / веса | ✅ | 🟡 индексы есть, веса нет | малый |

**Вывод по данным:** для рынка РФ покрытие близко к промышленному по ядру
(ставки/волы/кредит/commodity/equity/FX). Главные разрывы — реальное время,
reference data полнота, факторные библиотеки.

## 4. Список дальнейших улучшений (приоритизирован)

### P1 — представления поверх готовых движков (высокая ценность, дёшево)
1. **Risk Decomposition view** — разложение портфельного риска/VaR по факторам,
   секторам, бумагам (presenter + heatmap/bar; данные факторов уже есть).
2. **Interactive What-If / Scenario Builder** — слайдеры шоков (spot/rate/vol/FX)
   → full_reprice_pnl в реальном времени (движок full-reprice уже готов).
3. **XVA Dashboard** — EPE/ENE/PFE-профили + CVA/DVA/BCVA на графиках
   (risk/exposure.py готов, нужен таб с графиками).
4. **P&L Attribution view** — Greeks-разложение (pnl_explain готов) как waterfall.
5. **VaR Backtest tab** — Kupiec/Christoffersen + exceptions-график (движок готов).
6. **Drill-down** портфель → позиция → cashflow (bondization уже в БД).

### P2 — аналитика и атрибуция
7. **Factor model** (Barra-style): риск-факторы из истории, β-экспозиции,
   систематический/идиосинкратический риск.
8. **Scenario library** именованных исторических шоков (2022/2008 окна — история есть).
9. **Что-если по кривой** (KRD-шоки по тенорам, твисты) на реальной КБД.
10. **Liquidity/concentration** метрики по портфелю (объёмы из equity_quotes есть).

### P3 — данные и платформа
11. **Intraday refresh** — переход с EOD на периодический intraday-снапшот
    (ISS отдаёт текущее; нужен планировщик чаще EOD).
12. **Reference data** полнее: corporate actions, рейтинги АКРА/ЭкспертРА (скрейпинг).
13. **AI auto-commentary** — LLM-нарратив по портфелю/риску/рынку (тренд Aladdin;
    дёшево через API, наше governance даёт надёжный контекст).
14. **Persistence/таймсерии снапшотов** для трендов риска во времени.

### P4 — направления роста
15. **Регуляторный модуль** (если потребуется): SA-CCR, FRTB-SA, нормативы ЦБ РФ.
16. **ESG/climate-риск** (MARS Climate как ориентир).
17. **Hedge accounting** (эффективность хеджа).
18. **Мультивалютная консолидация** портфеля.

### Что НЕ копировать (осознанно)
Real-time tick-стриминг и глобальное межассетное покрытие — вне реалий
бесплатных данных и десктоп-архитектуры; новости/чат — не профиль quant-движка.

---

## 5. Стратегический вывод

RiskCalc по **ядру прайсинга, рыночным данным РФ и explorer-визуализации**
сопоставим с промышленными платформами; **governance/audit/identity-first
валидация — наше отличие** (у многих это внешний слой). Самый дешёвый рост —
**P1: представления поверх уже валидированных движков** (декомпозиция риска,
what-if, XVA/атрибуция-дашборды). Это даёт «институциональный» вид без новых
численных моделей. Реальное время и регуляторика — стратегические ставки под
конкретного пользователя (банк vs брокер vs инвесткомпания).
